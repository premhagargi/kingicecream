# **App Name**: King Ice Cream

## Core Features:

- Immersive Hero Section: A full-viewport hero with large product visuals, minimal headline, and subtle motion effects (fade, scale) on load for an immediate premium impression.
- Product Showcase & Interaction: Displays ice cream products via flexible grid or horizontal scroll, focusing on high-resolution images, premium descriptions, and fluid hover animations (scale, shadow, slight motion).
- Generative Ingredient Storytelling Tool: An AI tool to craft evocative, short narratives for unique ingredients, craftsmanship details, or origin stories, fitting the 'minimal text, maximum impact' and luxury aesthetic.
- Cinematic Scrolling & Transitions: Implements smooth, scroll-triggered animations (fade-in, slide-up, slight scale) for seamless page progression, making each section feel like a scene in a brand film.
- Dynamic Brand Narratives: Dedicated sections featuring bold typography and minimal copy to emphasize luxury and craftsmanship, including a repeating horizontal ticker for news or announcements.
- Collection/Category Navigation: Horizontally scrollable categories (e.g., 'Gelato', 'Ice Pops') with smooth snapping interactions for elegant exploration of product lines.
- Fullscreen Overlay Navigation Menu: A premium, fullscreen overlay menu with a dynamic left-side visual preview that updates on menu item hover, and a right-aligned vertical navigation list, designed for immersive exploration.

## Style Guidelines:

- Primary color: Black (#000000). Used for striking contrasts, elegant typography, and key interactive elements to create a sophisticated, high-impact aesthetic.
- Background color: Pure White (#FFFFFF). A pristine, clean canvas that allows product imagery and black text to stand out, embodying luxury and minimalist design.
- Accent color: Various shades of Grey (e.g., #333333, #CCCCCC). Used sparingly for subtle differentiation, hover states, or secondary textual elements to add depth without breaking the monochromatic theme.
- Headline font: 'Bebas Neue'. A strong, condensed sans-serif for impactful primary headlines, conveying a modern and bold luxury feel.
- Body font: 'Inter'. Chosen for its clean, modern, and highly legible characteristics, this sans-serif font ensures readability and a contemporary feel for all body text.
- Supplemental font: 'Geist'. Utilized for subheadings, experimental text elements, or specific UI components to add a modern, tech-forward touch, aligning with the brand's experimental aesthetic.
- Utilize minimalist and clean line-art icons with ample whitespace, ensuring they complement the elegant typography and allow the high-resolution product imagery to dominate visually.
- Implement a fullscreen, editorial-style layout with asymmetric yet balanced grid sections. Prioritize generous spacing and cinematic transitions between elements and sections to create a visual-first storytelling experience. Strictly avoid rounded corners anywhere in the app.
- Employ smooth, scroll-triggered animations for elements (fade-in, slide-up, slight scale) and fluid transitions for hover states across all clickable elements, ensuring motion feels premium and cinematic, never jarring.
